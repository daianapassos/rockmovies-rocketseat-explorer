## 🛠 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- ReactJS
- ViteJS
- Styled Components
- React Router

## ⚙️ Utilização

1. selecione a pasta do projeto: ``` cd rocketmovies ```
2. instale as dependências com: ``` npm install ```
3. inicie o projeto com: ``` npm run dev```
